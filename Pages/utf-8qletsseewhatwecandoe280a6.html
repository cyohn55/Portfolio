But, not all is doom and gloom with the help of AI. I wanted to create a simple way where I could quickly add new posts to my portfolio without needing to code them by hand. I knew there were plenty of blogging tools available to me, but I wanted to attempt “vibe coding” to get a better understanding of what LLMs are capable of.

How it works: For the blog tool, I wanted the ability to draft a new email, include text, images, and video in the body of the email, which would act as an HTML blueprint for building a new web page. The email would trigger a git push command, pushing my portfolio website live to my github pages website, once the email is sent.
